## RedProtect Add-on: RedBackups

Creates backups of region `.mca` files for RedProtect region locations.

### Commands

- Reload: `/redbackups reload` or `/rbkp reload`
- Manual backup: `/redbackups backup [here]` (use `here` to back up the current location)

### Permissions

- Reload: `redbackup.cmd.reload`
- Backup: `redbackup.cmd.backup`
